== Description ==

PulseForms is a lightweight and customizable WordPress form builder plugin created for users who want clean forms, strong customization, secure submissions, and beautiful frontend styling without locking core features behind a paid version.

PulseForms includes form creation, shortcode embedding, styled frontend forms, email notifications, submission storage, file uploads, custom captcha, honeypot protection, rate limiting, error logs, and admin-friendly submission viewing.

The plugin is designed to be simple, clean, extensible, and useful for real WordPress websites.

== Shortcode ==

Use this shortcode to display a form:

[pulseform id="1"]

Replace 1 with the ID of your form.

== Installation ==

1. Download the plugin ZIP file.
2. Go to WordPress Admin.
3. Navigate to Plugins > Add New.
4. Click Upload Plugin.
5. Upload pulseforms.zip.
6. Activate the plugin.
7. Go to PulseForms in the WordPress admin menu.
8. Create your first form.
9. Copy the shortcode into any page, post, or page builder.

== Frequently Asked Questions ==

= Does PulseForms store submissions? =

Yes. Submissions can be stored in the WordPress database and viewed from the PulseForms admin area.

= Can I disable submission storage? =

Yes. Each form includes a setting to enable or disable saved submissions.

= Does PulseForms send emails? =

Yes. PulseForms can send admin notification emails and user confirmation emails for supported form types.

= Are login and registration forms treated differently? =

Yes. Login and registration forms do not enable normal submission emails by default.

= Does PulseForms support file uploads? =

Yes. File uploads are supported with allowed file type and maximum size settings.

= Does PulseForms include spam protection? =

Yes. PulseForms includes honeypot protection, custom captcha, and basic rate limiting.

= Can I customize the form design? =

Yes. Forms can use PulseForms styling, inherit theme styling, use preset themes, custom colors, radius controls, and custom CSS.

= Does PulseForms use Font Awesome? =

No. PulseForms uses Material Icons only.
